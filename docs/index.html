Features

見た目は自由、動きはシンプル。

公開デモの配色ルールと同じ bg-color-* を前提にしているので、導入時の見た目合わせがしやすく、 既存 UI にもそのまま馴染ませやすい設計です。

01

依存なしで動作

フレームワークに縛られず、HTML と少量の初期化コードだけで導入できます。小さな導線 UI に向いた構成です。

02

背景色と同期できる

インジケータだけでなく `bgArea` にもクラスが付くため、ページ全体やセクション単位の配色変化もまとめて扱えます。

03

CSS で仕上げやすい

色、hover、active 表現は CSS 側で素直に調整できます。今回のページもそのルールでデモを構成しています。

Quick Start

セットアップは 3 ステップです。

1

インストール

npm install circle-navi
2

HTML を用意

<div class="navi__inner">
  <span class="circle bg-color-1"></span>
  <button data-index="0" class="inview">01</button>
  <button data-index="1">02</button>
  <button data-index="2">03</button>
</div>
3

JavaScript で初期化

import { CircleNavi } from "circle-navi";

const menu = new CircleNavi({
  btn: ".navi__inner button",
  target: ".circle",
  bgArea: "body",
  diameter: 40,
  interval: 5,
});

menu.addEvent();

Options

主要オプション

Option Type Description
btn string 操作対象ボタンの CSS セレクタ。各ボタンに data-index が必要です。
target string 動かすインジケータ要素の CSS セレクタです。通常は .circle を指定します。
bgArea string ボタン切り替えに合わせて、指定した要素へ bg-color-1 から bg-color-n のクラスを付け替えるための CSS セレクタです。
diameter number インジケータ直径です。このページでは 40 を使っています。
interval number ボタン間のギャップ幅です。CSS の gap と同じ値に揃えると自然に見えます。
